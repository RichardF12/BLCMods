			<category name="Description">
				<comment>By Aaron0000</comment>
				<comment>Features:</comment>
				<comment>Changes the Jakobs, Maliwan, and Torgue pistols to behave like the Revolvers of Borderlands 1.</comment>
				<comment>High damage.</comment>
				<comment>High accuracy.</comment>
				<comment>High critical hit damage (+50% on Torgue, +100% on Maliwan, +150% on Jakobs.)</comment>
				<comment>Low fire rate.</comment>
				<comment>High recoil.</comment>
				<comment>Consumes 3 ammo per shot (Maliwan consumes 4 per shot.)</comment>
				<comment>Averages six shots a clip (five for Maliwan).</comment>
				<comment>Double accessory is turned into a Masher accessory that only "mashes" the Revolvers, is a normal Double accessory otherwise.</comment>
				<comment>Accuracy accessory is turned into an accessory that behaves like Mag5 from Borderlands 1 (High damage, two-shots before reloading. Slower reloads, tighter crosshairs). Behaves like a normal accuracy accessory for non-Revolvers.</comment>
				<comment>NOTES;</comment>
				<comment>Certain special guns have been given adjustments to better fit the changes caused by revolving the Jakobs/Maliwan/Torgue pistols. Disable if needed (such as for mods that go over those items). The items in question are;</comment>
				<comment>All Unique/Legendary/Pearlescent Jakobs pistols except for the Law (Maggie was adjusted so that the projectile count as a masher is x24 instead of x35.)</comment>
				<comment>Maliwan E-Tech pistols (just the ammo cost), Infection, and (Unofficial Community Patch) Wanderlust (UCP Wanderlust was adjusted so that the projectile count as a masher is x14 instead of x18.)</comment>
				<comment>All Unique/Legendary/Seraph Torgue pistols (Unkempt Harold was adjusted so that the projectile count as a masher is x5 instead of x6.)</comment>
				<comment>Thanks to Koby (modified code) and Hemaxhu (original code), Jakobs pistols have had their animations adjusted to better fit the feel of a slow-powerful gun, instead of fanning the hammer.</comment>
				<comment>FIXES FOR OTHER MODS;</comment>
				<comment>If you want to use the standalone version of mashers (that "mashes" all pistols, instead of just Jakobs/Maliwan/Torgue), just activate Mashers after this mod.</comment>
				<comment>If using my Plasma Pistols mod, there is a fix for the standard Maliwan e-tech pistols (or Jakobs and Torgue, if you have the mod for it) and the Wanderlust at the bottom.</comment>
				<comment>If using my Defiler mod, there is a fix for it at the bottom.</comment>
				<comment>If using my Discipline mod, there is a fix for it at the bottom.</comment>
				<comment>If using my Smasher mod, there is a fix for it at the bottom (to reduce the ammo cost).</comment>
				<comment>If using my 88 Fragnum mod, there is a fix for it at the bottom (to reduce the ammo cost).</comment>
				<comment>If using the Salvation from The Campin' Dutchpack mod, there is a fix for its damage (to compensate for the lack of critting) at the bottom.</comment>
				<comment>If using the Painbringer from The Campin' Dutchpack mod, there is a fix for it at the bottom.</comment>
				<comment>If using the Triple from The Campin' Dutchpack mod, there is a fix for each version of it at the bottom.</comment>
